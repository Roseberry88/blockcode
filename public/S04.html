Python 코드

def bubble_sort(arr): n = len(arr) swapped = True while swapped: swapped = False for i in range(1, n): if arr[i-1] > arr[i]: arr[i-1], arr[i] = arr[i], arr[i-1] swapped = True n -= 1 # 끝 원소는 이미 정렬됨 return arr

알고리즘 정보

버블 정렬 (Bubble Sort)

시간 복잡도: O(n²)

공간 복잡도: O(1)

안정 정렬: Yes

알고리즘 설명

버블 정렬은 인접한 두 원소를 비교하여 필요에 따라 위치를 교환하는 방식으로 동작합니다. 배열의 끝부터 정렬이 완성되며, 각 패스마다 가장 큰 원소가 올바른 위치로 "버블링"됩니다. 이 과정은 더 이상 교환이 필요 없을 때까지 반복됩니다.
